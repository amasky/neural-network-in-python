# Neural Network in Python
* 1 hidden layer NN in Python

## Dependencies
Python 2 or 3, NumPy, scikit-learn, matplotlib  

## Usage

train the model: 

```shellsession
➜  python train.py
```

classify digits with the trained model

```shellsession
➜  python predict.py
```
